#                                                                              #
# 1. Compile                                                                   #
# --------------                                                               #
#                                                                              #
# - Import the project into Eclipse                                            #
# - Clean project                                                              #
# - Compile project                                                            #
#                                                                              #
# 2. Create Jar                                                                #
# --------------                                                               #
#                                                                              #
# - First compile the project                                                  #
# - Export the project with Eclipse                                            #
# - Export as Executable Jar                                                   #
#                                                                              #
# 3. Start Game                                                                #
# --------------                                                               #
#                                                                              #
#    3.1. Via Eclipse                                                          #
#                                                                              #
#         - Open project in Eclipse                                            #
#         - Press the run button                                               #
#                                                                              #
#    3.2. Via Jar                                                              #
#                                                                              #
#         - Go to your export destination of the Jar file                      #
#         - Double click on the Jar file                                       #
#                                                                              #
# 4. Commands                                                                  #
# --------------                                                               #
#                                                                              #
#  <- (left arrow key)    =    Move tank to left                               #
#  -> (right arrow key)   =    Move tank to right                              #
#  ^  (up arrow key)      =    More acute shooting angle                       #
#  v  (down arrow key)    =    More shallow shooting angle                     #
#  _  (space hold)        =    Load shooting power                             #
#  _  (space released)    =    Fire                                            #
#                                                                              #
# 5. Goals                                                                     #
# --------------                                                               #
#                                                                              #
# - Destroy the enemy tank team                                                #
#                                                                              #
